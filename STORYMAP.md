## 4. Phase 1：ドメイン公開の手順

1. **ローカル確認** — 全ページ表示・リンク・レスポンシブをブラウザで確認
2. **ホスティング選定** — 静的サイト向け（Cloudflare Pages / Netlify / Vercel / GitHub Pages）
3. **デプロイ** — リポジトリ連携 or ドラッグ&ドロップで公開（`*.pages.dev` 等の仮URL発行）
4. **独自ドメイン** — 取得済みドメインを接続 or 新規取得（例: cherrymoon.jp）
5. **DNS設定** — ホスティングの指示に従い CNAME / A レコードを設定
6. **HTTPS / 仕上げ** — 証明書自動発行を確認、OGP・favicon・404ページを整備

> ※ ホスティングの選定とドメイン名は方針確認が必要（次のメッセージで質問）。
